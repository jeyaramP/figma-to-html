# Figma to Responsive HTML

A **pixel-perfect responsive HTML/CSS** implementation based on the provided Figma design.  
This project focuses on **clean, semantic markup**, **organized CSS**, and **mobile responsiveness** using modern layout techniques like Flexbox and Grid.

---

## Features

- Fully responsive layout for all screen sizes 
- Clean and semantic HTML structure  
- Organized and reusable CSS classes  
- Responsive Flexbox & Grid layout design  
- Cross-browser compatibility (Chrome, Firefox, Safari, Edge) 
- Smooth animations 
- Deployed on Vercel for live preview  

---

## Objective

Convert the given Figma design into a **responsive front-end layout** following the exact design specifications.  
Reference: [LeadCRM.io](https://www.leadcrm.io/)

---

## Tech Stack

- **HTML5** – Semantic markup structure   
- **CSS3** – Styling and responsive design 
- **CSS3**  
- **Bootstrap 5** – Responsive grid system
- **Media Queries** – Mobile responsiveness   
- **Flexbox & Grid** – Layout alignment and structure     
- **Deployment:** Vercel
